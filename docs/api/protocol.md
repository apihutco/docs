# 一句问候

基于时间、日期、季节、天气状况等因素返回一句问候

### 请求地址

`HTTPS` `GET`

[https://apihut.net/greet](https://apihut.net/greet)

### 请求参数

| 参数     | 必填 | 说明           |
| -------- | ---- | -------------- |
| time     | 否   | 时间 [代码列表](#time时间)       |
| day      | 否   | 星期 [代码列表](#day星期)       |
| weather  | 否   | 天气 [代码列表](#weather天气)       |
| language | 否   | 语言 [代码列表](#language语言)           |
| no_range | 否   | 不使用随机返回 |

### 响应结果

| 名称       | 类型   | 说明                           |
| ---------- | ------ | ------------------------------ |
| code       | int    | 状态码                      |
| msg        | string | 状态信息                       |
| data       | object | 天气结果                       |
| - word     | string | 词语                           |
| - sentence | string | 句子                           |
| - author   | string | 句子作者，没作者则不返回此字段 |

### 返回示例

```json
{
  "code": 200,
  "msg": "成功",
  "data": {
    "word": "傍晚好",
    "sentence": "来日方长，何惧车遥马慢"
  }
}
```
